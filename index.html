SQL

1. What does SQL stand for?

Structured Query Language

2. Which SQL command is used to retrieve data from a database?

SELECT

3. Which SQL clause is used to filter records in a SELECT statement?

WHERE

4. What does the SQL command "INSERT INTO" do?

Adds a new record

5. Which SQL statement is used to modify existing data in a database?

UPDATE

6. What is the purpose of the SQL command "DELETE"?

To remove data from a table

7. What does SQL normalization refer to?

Reducing redundancy in data

8. Which SQL function is used to find the maximum value in a column?

MAX()

9. What SQL command is used to change the structure of a table?

ALTER

10. Which SQL clause is used to sort the result set of a SELECT statement in ascending order?

ORDER BY
